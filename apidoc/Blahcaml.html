Module Blahcaml


module Blahcaml: sig .. end
The Blahcaml module defines functions for the conversion of TeX equations into MathML.
val init_dtd : unit -> unit
Initialisation of the MathML2 DTD. This function can take several seconds to complete, but only needs to be called once. If you don't worry about safety and only invoke Blahcaml.unsafe_mathml_from_tex, then you never need to run this function. On the other hand, if you call Blahcaml.sanitize_mathml or Blahcaml.safe_mathml_from_tex, and the DTD has never been manually initialised, then this function will automatically be invoked. Therefore, if you want predictability on the runtime performance of the sanitisation functions, you should invoke it during the initialisation stage of your programme.
val sanitize_mathml : string -> string
Given a string containing potentially unsafe MathML, this function makes sure the it conforms to the MathML2 DTD. If safe, the function returns the original string. If, however, the string does not conform to the DTD, an exception is raised. See the PXP documentation for the meaning of these exceptions. Note that if the DTD has never been initialised, this function will automatically invoke Blahcaml.init_dtd.
val unsafe_mathml_from_tex : string -> string
Converts a string containing an equation in TeX format into another string containing the same equation in MathML. No checking is done to make sure the result conforms to the MathML2 DTD! If that assurance is required please use the Blahcaml.safe_mathml_from_tex.
val safe_mathml_from_tex : string -> string
Converts a string containing an equation in TeX format into another string containing the same equation in MathML. The resulting string is checked to make sure it conforms to the MathML2 DTD. If it does not, an exception is raised. See the PXP documentation for the meaning of these exceptions. Note that if the DTD has never been initialised, this function will automatically invoke Blahcaml.init_dtd.
